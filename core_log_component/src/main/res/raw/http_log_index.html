DDPlayTV HTTP 日志管理

服务提供全量 ZIP、分段目录、最新分段、按时间点分段下载与手动清理，下载后请在本地使用文本工具（如 rg）检索排查。

日志分段目录

点击“刷新分段目录”查看当前可下载日志时段

下载接口：GET /api/v1/logs/download
最新分段：GET /api/v1/logs/latest-segment
时间点分段：GET /api/v1/logs/segment-at?timestampMs=<ms>
分段目录：GET /api/v1/logs/segments
清理接口：POST /api/v1/logs/clear
如果浏览器环境受限，也可使用 curl： curl -L -H "Authorization: Bearer <token>" -o ddplaytv-logs.zip http://<ip>:17010/api/v1/logs/download
curl -L -H "Authorization: Bearer <token>" -o ddplaytv-latest-segment.jsonl http://<ip>:17010/api/v1/logs/latest-segment
curl -L -H "Authorization: Bearer <token>" -o ddplaytv-segment-at.jsonl "http://<ip>:17010/api/v1/logs/segment-at?timestampMs=<ms>"
curl -L -H "Authorization: Bearer <token>" "http://<ip>:17010/api/v1/logs/segments"
